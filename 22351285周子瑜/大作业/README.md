# Animation
Unity 3d 实现真实场景泥石流灾害效果

1、虚拟地形场景的构建
基于案例区域遥感影像数据、高程数据构建灾害虚拟地形环境场景。
2、地物场景建模
对真实地理场景中的房屋、树木、道路等三维地物进行建模。
3、泥石流灾害场景表达
采用泥石流灾害数值模拟计算结果，对灾害动态演变进行三维动态可视化展示。
