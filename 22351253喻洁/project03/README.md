# Weapon Display

该项目使用 C++ 和 OpenGL 创建描绘纹理武器模型的 3D 场景。库包括 GLEW、GLFW、GLM 和 stb_image。

### 控制

- WASD：移动相机
- Q/E-Spc/Ctrl：向上/向下
- 鼠标：俯仰/偏航
- 滚动：移动速度
- 1 / 2：提高/降低相机速度
- P：透视/正交切换
- L：灯开关切换
